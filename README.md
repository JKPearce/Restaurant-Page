# Restaurant Page
A single page project for learning about JavaScript ES6 modules using imports and exports

Live Demo: [Here](https://jkpearce.github.io/Restaurant-Page/)

## Goals
The goal in this project is to understand modules by using npm and webpack

### Tech used 
- HTML
- SCSS
- JavaScript
- NPM
- WebPack

#### Future improvements
- Add unique animations on page swaps
- Clicking on menu item brings up nutrition details
- Add menu filter to filter to certain food types e.g. chicken, beef, pasta, pizza, etc.
- Add email form on contact page
